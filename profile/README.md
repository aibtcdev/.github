## Repository Map

| Repository         | Domain             | Description                                                                                                                          |
| ------------------ | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| [landing-page](https://github.com/aibtcdev/landing-page)       | [aibtc.dev](https://aibtc.dev)          | Main website for https://aibtc.dev                                                                                                   |
| [aibtcdev-backend](https://github.com/aibtcdev/aibtcdev-backend)   | [core.aibtc.dev](https://core.aibtc.dev)     | CrewAI powered agents with Bitcoin and Stacks tooling.                                                                               |
| [agent-tools-ts](https://github.com/aibtcdev/agent-tools-ts)     |                    | Agent tooling powered by Bun and Stacks.js, used by aibtcdev-backend as a submodule                                                                                      |
| [aibtcdev-frontend](https://github.com/aibtcdev/aibtcdev-frontend)  | [sprint.aibtc.dev](https://sprint.aibtc.dev)   | Create, schedule, run, and chat with crews of agents that have their own wallets. Create and manage DAOs, trade, transfer, and more! |
| [aibtcdev-services](https://github.com/aibtcdev/aibtcdev-services)  | [services.aibtc.dev](https://services.aibtc.dev) | Infrastructure services for aibtcdev projects                                                                                        |
| [aibtcdev-contracts](https://github.com/aibtcdev/aibtcdev-contracts) |                    | Clarity smart contracts and test suite                                                                                     |
| [aibtcdev-cache](https://github.com/aibtcdev/aibtcdev-cache)     | [cache.aibtc.dev](https://cache.aibtc.dev)    | Queries and caches external APIs with custom scheduling and rate-limiting, provides a drop-in replacement for aibtcdev projects      |
| [aibtcdev-docs](https://github.com/aibtcdev/aibtcdev-docs)       | [docs.aibtc.dev](https://docs.aibtc.dev)      | Gitbook documentation for using CrewAI AI agents with a Bitcoin wallet                                                                            |
